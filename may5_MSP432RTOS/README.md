# Real-time programming with Energia MT  
A tutorial about how to do real-time programming on Energia MT platform. 

## Requirements
* MSP432 Launchpad + USB cable.
* Energia 18

## Examples
1. **demo00** new task.
2. **demo01** task and ISR.
3. **demo02** semaphore.
4. **demo03** AT command example.
